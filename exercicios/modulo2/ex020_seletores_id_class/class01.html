Diferença entre ID e CLASS

ID é usado individualmente, ou seja, para um único elemento.
Class é utilizado em grupo, ou seja, para vários elementos que irão compartilhar a mesma característica

Por exemplo

Básico 1

Básico 1

Nesse caso ambos os títulos básico 1 são da mesma class e por isso herdaram as mesmas características.

Uma observação importante: É possível um mesmo elemento possuir mais de uma classe

Assim como também é possível um elemento possuir um ID e uma Class
