Principais formatações

Negrito / Destaque

Nesta frase temos um texto em negrito utilizando a tag <b> que é uma tag não semântica.

Nesta outra frase há um texto em negrito utilizando a tag <strong> que é uma tag semântica.

Itálico / Ênfase

Nesta frase temos um texto em itálico utilizando a tag <i>

Nessa frase temos um texto em itálico (em ênfase) com a tag <em>

Texto Marcado

É possível criar um texto marcado utilizando a tag <mark>

É possível adicionar CSS dentro de uma tag HTML, para realizar estilização como mudar a cor do texto marcado utilizando um style = "" dentro da tag HTML.

Para poder personalizar todos os <mark> existentes no código, é necessário criar um <style> no head, estilizando assim todas as tags

Texto grande e pequeno

Para criar um texto grande utiliza-se a tag <big> que atualmente está obsoleta, e para criar um texto pequeno utiliza-se a tag <small>

Texto deletado / riscado

É possível marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado. Com a tag <del>

Texto inserido / sublinhado

É possível marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois. Com a tag <ins>

Texto sobrescrito

Para inserir um texto sobrescrito utilizado em x2+2x = 0 basta utilizar a tag <sup>

Texto subscrito

Para inserir um texto subscrito utilizado em H2O basta utilizar a tag <sub>
